# SKY INVADERS
*A modification of the classic Space Invaders*

## What's New?
The player and the enemies are now jets, flying above a battlefield. The objective is to survive for as long as possible, with the difficulty increasing every 2 waves destroyed. 

* **Full Control:** Feel free to fly sideways, forward, or backward—you have full control!
* **Power-ups:** Every 1000 points, a star appears. It will give you a bonus to help you survive. 
* Careful! Don't get toasty!

## How do I play?
1. On the right side of this GitHub page, look for the **Releases** section.
2. Click on the latest release to get the most recent changes.
3. Download the `.exe` file (or the `.zip` file) and extract it.
4. Run the executable. 
*(Note: If Windows Defender blocks it, click on "More info" and then "Run anyway" to allow the execution).*
